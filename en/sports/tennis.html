Last Update: 19/01/2023

General Rules
  • All bets will be considered valid only when the match or specified period is played to completion unless otherwise stated in the individual Bet Type rules.
  • If the scoring format of a match differs from what was originally indicated, all bets will be void.
  • If there is a sudden change of venue (e.g., indoor to outdoor or vice versa) or court surface, before or during a match, all bets will be valid.
  • If a match starts before schedule only the transactions before the first point of the match will be considered valid. Transactions after the first point of the match will be considered invalid. (Except in-play bet types)
  • A ‘Super Tie-Break’ is considered as a game, for settlement purposes.
Bet Types

Winner

  • Predict who will win the match.

Set Handicap

  • Predict who will win the match with the indicated set handicap applied.

Game Handicap

  • Predict who will win more games in a specified period (e.g. Match, 1st Set, 2nd Set) after applying the indicated handicap.

Game Winner (Winner – Set X Game X)

  • Predict who will win a specific game of a specific set (e.g. Set 1 Game 1).

    • Game at 15/30 - Set X, Game Y

    • Predict whether the specified game (e.g., Set 1, Game 1) will end with either player winning the game, while their opponent scores only 15 or 30. E.g., Win-15, Win-30 or 15-Win, or 30-Win

    Total Points: Over / Under - Set X, Game Y

    • Predict whether the total number of points played in a specified game (e.g., Set 1, Game 1) will be over or under the indicated line. E.g., 4-0 (Win-0), 4-1 (Win-15), 4-12 (Win-30).

    Total Games: Over / Under

    • Predict whether the total number of games played in a specified period (e.g. Match, 1st Set, 2nd Set) will be over or under the indicated total line.

    Total Games: Odd / Even

    • Predict whether the total number of games played in a specified period (e.g. Match, 1st Set, 2nd Set) will be odd or even.

    Set Betting Markets

    • Predict the outcome of the specified set.

    Match Correct Score

    • Predict the final sets score of the match.

    Total Points - Over / Under

    • Predict whether the total number of points won by a player or pair of players in a match will be over or under the indicated total line.
